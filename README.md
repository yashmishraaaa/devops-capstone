# DevOps Capstone Project 🚀

## Features
- Dockerized Node.js App
- Kubernetes Deployment
- CI/CD with GitHub Actions
- Monitoring with Prometheus & Grafana
- Security using Trivy

## Steps
1. Run app locally
2. Build Docker image
3. Push to DockerHub
4. Deploy to Kubernetes
5. Setup monitoring
